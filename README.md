# BBMP Advertisement Licence - Login Form

🛡️ A simple, responsive login form designed for the Bruhat Bengaluru Mahanagara Palike (BBMP) Advertisement Licence system.

---

## 📋 Features

- Clean, modern login UI
- Fully responsive layout
- BBMP logo and bilingual branding (Kannada + English)
- Styled form with validation-ready inputs
- Minimal dependencies, uses pure HTML/CSS
- Easy to integrate into existing BBMP systems

---
